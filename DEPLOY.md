# Публикация сайта SKYWAY

## Перед выкладкой на хостинг

1. **Ключ DeepSeek API (терминал Смита)**  
   В `index.html` найдите строку:
   ```html
   <script>window.DEEPSEEK_API_KEY = '';</script>
   ```
   Подставьте свой ключ (получить: [DeepSeek Platform](https://platform.deepseek.com)), чтобы ИИ-агент в блоке «Терминал» отвечал на запросы.  
   Если оставить пустым, при ошибке будет показано сообщение «КАНАЛ ПЕРЕГРУЖЕН».

   **Безопасность:** ключ в HTML виден в коде страницы. Для продакшена предпочтительно проксировать запросы к DeepSeek через свой backend.

2. **Formspree (форма заявки)**  
   Форма на главной отправляет данные на `https://formspree.io/f/xnqkvnny`. Убедитесь, что этот Formspree-формы привязан к вашему email, или замените на свой endpoint в атрибуте `action` формы с `id="contactForm"`.

3. **Домен**  
   В коде используются абсолютные URL вида `https://skyway-core.ru/`. Если домен другой, замените в:
   - всех `og:image` и `twitter:image` (полный URL картинки);
   - `sitemap.xml` (тег `<loc>`);
   - `robots.txt` (Sitemap);
   - JSON-LD в `<script type="application/ld+json">` (url, logo, sameAs и т.д.).

## После публикации

- Проверьте работу формы (отправка и приход писем).
- Откройте главную и блок «Терминал» — введите текст и нажмите Enter (при указанном ключе должен прийти ответ).
- Убедитесь, что несуществующая страница открывает `/404.html`.
- При необходимости подключите Яндекс.Метрику или Google Analytics.
